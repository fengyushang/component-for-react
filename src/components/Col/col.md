## API

```html
<Col md={4} lg={3}> </Col>
```

| 参数 | 说明 | 类型 | 默认值 | 必需 |
| --- | --- | --- | --- | --- |
| span | 栅格占位格数(1-12) | number | - | 否 |
| xs | <768px(1-12) | number | - | 否 |
| sm | >768px(1-12) | number | - | 否 |
| md | >992(1-12) | number | - | 否 |
| lg | >1200(1-12) | number | - | 否 |
| xl | >1400px(1-12) | number | - | 否 |
