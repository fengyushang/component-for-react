## API

```html
<Row align="top" justify="start" gutter={8}> </Row>
```

| 参数 | 说明 | 类型 | 默认值 | 必需 |
| --- | --- | --- | --- | --- |
| align | 垂直对齐方式：top middle bottom | string | - | 否 |
| justify | 水平排列方式：start end center space-around space-between | string | start | 否 |
| gutter | 栅格间隔(px) | number | 0 | 否 |
