## API

```html
<Tipsy className={className} tipsyText='text' placement="top"> </Tipsy>
```

| 参数 | 说明 | 类型 | 默认值 | 必需 |
| --- | --- | --- | --- | --- |
| className | 类名设置：className | string | - | 否 |
| tipsyText | 悬浮窗文本内容：text| string | - | 是 |
| placement | 悬浮窗位置：top right bottom left | string | top | 否 |
